Logg inn for å hente laget ditt

Logg inn

Denne løsningen bruker Firebase Authentication (e-post/passord) og Realtime Database. Brukeren får tilgang til sitt lag via /users/{uid}/teamId.

Opprett bruker

Tips: Sett teamId manuelt i Realtime Database, eller lås dette til administrator senere.

Hva appen kan gjøre

  • Logg inn og hent riktig lag automatisk
  • Sett opp spillere, trøyenummer, posisjon og hvem som er tilgjengelig
  • Start kampklokke, pause, 2. omgang og stopp
  • Registrer mål, assist, bytter, kampnotater og hendelser
  • Autogenerert kampreferat i «AI-stil» etter kampslutt
  • Sesongrapport: minutter, kamper, mål, assist og bidrag
  • Eksport til CSV og utskrift fra mobilen

Firebase-oppsett

For sikker drift bør ekte AI-rapport lages via Cloud Functions eller annen backend. Denne mobilprototypen lager derfor en lokal, automatisk oppsummering uten å eksponere API-nøkler i klienten.

Foreslåtte ekstra ting (lagt til)

  • Assist-føring i tillegg til målscorer
  • Spilletid per spiller via byttelogikk
  • Tilgjengelig/borte/skadet-status
  • Notater per kamp og eksportmulighet
  • Sesongoversikt med bidragsscore (mål + assist + minutter + kamper)
